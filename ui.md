# UI System

---

## Convention
 
- 사용자가 한번에 이해할 수 있도록 객관적인 사실만 기술하여야 한다.
- 각 요청에 맞는 메서드를 사용하여야 한다.

---



### ConsoleUtils

* logInfo() : 요청 성공, 정보 제공을 위한 정보 출력용 메소드


* logWarn() : 요청 실패와 같은 경고 출력용 메소드


* logError() : 예외와 같은 시스템 오류 출력용 메소드


### UserInterfaceUtils

* uiTitle() : 각 메뉴 접속시 사용되는 타이틀 메소드
